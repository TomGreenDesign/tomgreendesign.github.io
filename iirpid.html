Infinite Impulse Response (IIR) Parameter Calculator for PID Controllers

This tool is designed to aid in the design of IIR filters to implement PID control on discrete data as measured from real-world systems.

Alebraic Solution

The function of a PID controller can be described in the time-continuous Laplace domain by the following transfer function: $$ C_{pid(s)} = K(1+{1 \over T_i s}+T_d s) $$ However, digital systems operate in the discrete time domain, whereby samples or snapshots of the measurand amplitude are taken at regular time intervals, equal to the inverse of the sampling frequency: $$T = {1 \over F_s} $$ Because the measured data is quantised, we substitute a bilinear transform into the laplace variable \(s\) in the transfer function: $$ s=({2 \over T} {z-1 \over z+1})$$ Such that $$ C_{pid(z)} = K+{KT(z+1) \over T_i (z-1)}+ {2K(z-1) \over T(z+1)}. $$ By rearranging for a common denominator, $$ C_{pid(z)} = {{2KTT_i(z+1)(z-1) + KT^2(z+1)^2+4KT_iT_d(z-1)^2} \over {2TT_i(z+1)(z-1)}} $$ And by normalising for \(z^2\) in the denominator, $$ C_{pid(z)} = { { {K \over TT_i} TT_i(z+1)(z-1) + KT^2(z+1)^2 + 4KT_iT_d (z-1)^2} \over {z^2 -1} } $$ Which is then expanded to $$ C_{pid(z)} = { { (K+{KT \over T_i} + {4KT_d \over T})z^2 + ({2KT \over T_i}-{4KT_d \over T })z + ( {KT \over T_i} + {4KT_d \over T} - K) } \over {z^2-1} } = {Y(z) \over X(z)} $$ \(X(z)\) and \(Y(z)\) in the equation above are the inputs and outputs to the system, since the transfer function defines a ratio between output and input amplitudes, we can now cross-multiply to find the output amplitude \(Y(z)\) for a given input amplitude \(X(z)\) in the \(z\) domain. $$ z^2Y(z) = {(K + {KT \over T_i} + 4KT_d \over T)z^2X(z) + ({2KT \over T_i} - {4KT_d \over T})zX(z)}$$ $$ + ({KT \over T_i} + {4KT_d \over T} -K)X(z) + Y(z) $$ In the bilinear transform, each power of \(z\) represents a unit time delay of period \(T\), and as such we can convert these orders of \(z\) into array indices of sampled data, since these samples are taken at a fixed sampling rate \(F_s\). $$ Y(n+2) = (K+{KT \over T_i} + {4KT_d \over T})X(n+2) + ({2KT \over T_i } - {4KT_d \over T})X(n+1) + $$ $$({KT \over T_i} + {4KT_d \over T} - K)X(n) + Y(n) $$ Where \(Y\) is an array of output data and \(X\) the array of input samples. The above equation calculates the output two samples in the future; this is not helpful to us, since we would require input samples from the future also! To resolve this, we can simply offset the array indices by two. It must be noted, however, that the first two sample will be calculated with the wrong value of \(Y(n)\) and that this error with propagate through the first several cycles of the function. $$ Y(n) = (K + {KT \over T_i} + {4KT_d \over T})X(n) + ({2KT \over T_i} - {4KT_d \over T})X(n-1) +$$ $$ ({KT \over T_i} + {4KT_d \over T} - K)X(n-2) + Y(n-2) $$
